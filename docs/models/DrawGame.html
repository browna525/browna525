DrawGame

  • id : String
    The draw game ID (e.g. \"powerball\").
  • name : String
    The draw game name.
  • results : OneOfDrawGameResults
    Limited to the most recent applicable winning numbers. There may be multiple in the case of Pick games.
  • nextJackpot : String
    The next jackpot amount (e.g. \"$20 Million\").
  • nextDrawDate : date
    The next drawing date (e.g. \"2021-12-31\").
  • expiringTickets : List
    An array of unredeemed tickets that will expire soon.
  • recentWinners : List
    If part of the API, include the most recent winners.
  • updatedAt : Date
    The date and time this information was last updated.
