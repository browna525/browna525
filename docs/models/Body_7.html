  • game-names : List
    List of game names as comma separated values. If not informed all draws for all games will be returned
  • date-from : String
    Starting date to be used by date span draw searches
  • date-to : String
    Ending date to be used by date span draw searches
  • previous-days : String
    Amount of previous days from today to be used by day offset searches
  • next-days : String
    Amount of next days from today to be used by day offset searches
  • start-draw : String
    Starting draw number
  • end-draw : String
    Ending draw number
  • status : String
    Draw status to restrict results to
  • size : String
    Size of the elements in the page to be returned
  • lang : String
    Language in which to return names
  • selections : String
    A comma separated list of winning selections to be used to filter draw result searches
  • secondary-selections : String
    A comma separated list of winning secondary selections to be used to filter draw result searches
  • bet-type : String
    The bet type to used to filter drawresults searches, along with selections
  • exclude-prize-tiers : String
    If true, do not include prize tier information in response
