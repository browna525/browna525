Retailer

  • id : Integer
    Internal ID.
  • name : String
    Retailer name or chain (e.g. \"7-Eleven\").
  • address : String
    Street address.
  • address2 : String
    Street address continued.
  • city : String
    Retailer city.
  • zip : String
    Retailer ZIP code.
  • phone : String
    Retailer phone number.
  • lat : BigDecimal
    Geolocation latitude.
  • lng : BigDecimal
    Geolocation longitude.
  • hasVendingMachine : Boolean
    Is there a ticket vending machine on-site?
