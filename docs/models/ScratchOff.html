ScratchOff

  • id : Integer
    The game number (e.g. \"1056\").
  • name : String
    The game name.
  • price : Integer
    Ticket sales price in dollars.
  • launchDate : date
    The date the ticket launched.
  • endDate : date
    The date the ticket sales will conclude.
  • redemptionDate : date
    The last date available to redeem tickets.
  • overallOdds : String
    Overall odds for the ticket (e.g. \"1-in-4\").
  • prizes : List
    An array of prize tier information.
  • expiringTickets : List
    An array of unredeemed tickets that will expire soon.
  • recentWinners : List
    If part of the API, include the most recent winners.
  • winnerReport : String
    A URL to the winner report PDF.
  • updatedAt : Date
    The date and time this information was last updated.
