GameResults

  • id : Integer
    Internal ID.
  • date : date
    The draw date.
  • numbers : String
    The winning number string (e.g. \"1-2-7-52-61 PB4 X3\").
  • game : DrawGame
  • subtype : String
    An optional subtype (e.g. ezmatch, doubleplay, midday, evening, etc).
  • didRollover : Boolean
    Did the prize rollover to the next drawing, or rolldown to the next tier?
  • location : String
    Some jackpots include the winning ticket state(s).
  • video : String
    Typically a YouTube URL of the drawing video.
  • results : Object
    A schemaless free-for-all of prize data based on the game.
