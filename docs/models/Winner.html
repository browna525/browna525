Winner

  • id : Integer
    Internal ID.
  • name : String
    The winner's full name.
  • location : String
    The winner's location (including state if out-of-state).
  • prizeAmount : String
    The prize amount (e.g. \"$20 Million\").
  • date : date
    Date the prize was claimed (or won?).
  • image : String
    URL of the winner's photo.
  • game : OneOfWinnerGame
    The game the winner played.
  • pressRelease : News
