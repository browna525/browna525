ResultDTO

  • primary : String
    List of main numbers or different characters.
  • secondary : String
    List of additional numbers or different characters.
  • tertiary : String
    List of tertiary numbers or different characters.
  • multiplier : Integer
  • primaryRevealOrder : String
    Reveal order of main numbers or different characters
  • secondaryRevealOrder : String
    Reveal order of additional numbers or different characters.
  • tertiaryRevealOrder : String
    Reveal order of tertiary numbers or different characters
  • drawType : String
    Type of the draw, such as normal or bonus.
  • prizeTierId : String
    The prize tier these results are for. Typically only used with bonus or raffle type results.
