getGameResults

With no query params, list the most recent game results for all Draw Games. Otherwise restrict to date, date range, game, and/or winning numbers. Returns 400 error if provided query params are invalid.

URL

https://virtserver.swaggerhub.com/browna525/NumbersSearch/1.0.0/draw-results

HTTP Method

GET

Response Type

inline_response_200_2

Parameters

  • Query: date date

    Search results by date.

  • Query: dateFrom date

    Search results by date range (from).

  • Query: dateTo date

    Search results by date range (to).

  • Query: drawGameId String

    Restrict to a single game with draw game ID (e.g. \"powerball\").

  • Query: numbers String

    Ordered comma separated list of numbers to search.

  • Query: page

    Object of pagination related query params, depending upon the server pagination strategy.

  • Query: fields

    Return only certain model properties keyed by model type and specified by a comma-separated list of property names.
