This is the API

getRetailers

Retrieve a list of retailers ordered by distance from a user's location. The request will include either an address or latitude & longitude.

URL

https://virtserver.swaggerhub.com/browna525/NumbersSearch/1.0.0/retailers

HTTP Method

GET

Response Type

inline_response_200_7

Parameters

  • Query: address String

    User-provided address to be geocoded.

  • Query: lat BigDecimal

    Latitude provided by the user's browser.

  • Query: lng BigDecimal

    Longitude provided by the user's browser.

  • Query: radius BigDecimal

    Search radius in miles.

  • Query: page

    Object of pagination related query params, depending upon the server pagination strategy.

  • Query: fields

    Return only certain model properties keyed by model type and specified by a comma-separated list of property names.
